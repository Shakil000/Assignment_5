Important FAQ

What is the document object model(DOM) ?

The Document Object Model (DOM) is a platform-independent and language-neutral interface that organizes an HTML or XML document as a hierarchical tree structure. Each node in the tree represents a part of the document, with the DOM logically modeling the document in this tree format.

How do you select and element from the DOM?

We can use below five tags to select 1. getElementById(id). 2. querySelector(selector). 3. querySelectorAll(selector). 4. getElementsByClassName(className). 5. getElementsByTagName(tagName).

What is event delegation in the context of the DOM and why it is useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to manage events triggered by its child elements. Instead of attaching individual listeners to each child element, the parent listens for events that bubble up from its children. This method improves performance and simplifies code, especially when dealing with dynamic or large numbers of elements.
Event delegation is useful because,
Efficiency: Reduces memory usage by using fewer event listeners, especially when dealing with many child elements.
Handles Dynamic Elements: Easily manages elements that are added dynamically after the event listener is set up.

How do you manipulates an element's attributes and style using the DOM?

To manipulate an element's attributes and styles using the DOM, we can use various JavaScript methods to interact with HTML elements dynamically.
1. setAttribute(attribute, value) 2. getAttribute(attribute) 3. removeAttribute(attribute) 4. style.property 5. classList
